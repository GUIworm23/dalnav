About DalNav

DalNav is an application that assists users in finding and locating places on Dalhousie Studley campus, such as buildings, study areas, places to eat, residences, parking areas, bike racks, and bus stops. A user can input their class schedule, and be shown on a map which building their classes are located in, how long it will take to get to the building from their current location, and set up reminders for when to leave for class.

DalNav was created by Amina Abawajy, Ashlyn Campbell, Dylan McCrindle, Kyle Watson, and Osama Alarhabi.
